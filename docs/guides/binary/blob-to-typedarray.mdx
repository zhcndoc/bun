---
title: 将 Blob 转换为 Uint8Array  
sidebarTitle: "Blob 转 Uint8Array"  
mode: center  
---

[`Blob`](https://developer.mozilla.org/en-US/docs/Web/API/Blob) 类提供了多种方法来以不同格式读取其内容。此代码片段将内容读取为 `ArrayBuffer`，然后从该缓冲区创建一个 `Uint8Array`。

```ts
const blob = new Blob(["hello world"]);
const arr = new Uint8Array(await blob.arrayBuffer());
```
