title: 将 Uint8Array 转换为 DataView
sidebarTitle: "Uint8Array 到 DataView"
mode: center
---

[`Uint8Array`](https://developer.mozilla.org/en-US/docs/Web/JavaScript/Reference/Global_Objects/Uint8Array) 是一种 _类型化数组_ 类，意味着它是查看底层 `ArrayBuffer` 中数据的一种机制。以下代码片段创建了一个 [`DataView`] 实例，作用于与 `Uint8Array` 相同的数据范围。

```ts
const arr: Uint8Array = ...
const dv = new DataView(arr.buffer, arr.byteOffset, arr.byteLength);
```
