---
title: 将 ArrayBuffer 转换为 Buffer
sidebarTitle: "ArrayBuffer 转 Buffer"
mode: center
---

Node.js 的 [`Buffer`](https://nodejs.org/api/buffer.html) API 出现早于 JavaScript 语言中 `ArrayBuffer` 的引入。Bun 同时实现了这两者。

使用静态方法 `Buffer.from()` 从一个 `ArrayBuffer` 创建一个 `Buffer`。

```ts
const arrBuffer = new ArrayBuffer(64);
const nodeBuffer = Buffer.from(arrBuffer);
```

---

要创建一个仅查看底层缓冲区部分内容的 `Buffer`，请向构造函数传入偏移量和长度。

```ts
const arrBuffer = new ArrayBuffer(64);
const nodeBuffer = Buffer.from(arrBuffer, 0, 16); // 查看前16字节
```
