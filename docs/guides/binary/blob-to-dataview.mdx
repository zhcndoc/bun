---
title: 将 Blob 转换为 DataView
sidebarTitle: "Blob 转 DataView"
mode: center
---

[`Blob`](https://developer.mozilla.org/en-US/docs/Web/API/Blob) 类提供了多种方法，用于以不同格式读取其内容。此示例将内容读取为一个 `ArrayBuffer`，然后基于该缓冲区创建一个 `DataView`。

```ts
const blob = new Blob(["hello world"]);
const arr = new DataView(await blob.arrayBuffer());
```
