title: 将 Blob 转换为字符串
sidebarTitle: "Blob 转字符串"

---

[`Blob`](https://developer.mozilla.org/en-US/docs/Web/API/Blob) 类提供了多种方法以不同格式获取其内容，包括 `.text()`。

```ts
const blob = new Blob(["hello world"]);
const str = await blob.text();
// => "hello world"
```
