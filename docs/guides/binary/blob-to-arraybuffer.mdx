title: 将 Blob 转换为 ArrayBuffer
sidebarTitle: "Blob 转 ArrayBuffer"
mode: center
---

[`Blob`](https://developer.mozilla.org/en-US/docs/Web/API/Blob) 类提供了多种方法以不同格式使用其内容，其中包括 `.arrayBuffer()`。

```ts
const blob = new Blob(["hello world"]);
const buf = await blob.arrayBuffer();
```
