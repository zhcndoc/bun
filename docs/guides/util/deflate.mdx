---

使用 `Bun.deflateSync()` 对 `Uint8Array` 进行 DEFLATE 压缩。

```ts
const data = Buffer.from("Hello, world!");
const compressed = Bun.deflateSync("Hello, world!");
// => Uint8Array

const decompressed = Bun.inflateSync(compressed);
// => Uint8Array
```
