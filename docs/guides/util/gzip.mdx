title: 使用 gzip 压缩和解压数据
sidebarTitle: Gzip 压缩
mode: center
---

使用 `Bun.gzipSync()` 以 gzip 格式压缩 `Uint8Array`。

```ts
const data = Buffer.from("Hello, world!");
const compressed = Bun.gzipSync(data);
// => Uint8Array

const decompressed = Bun.gunzipSync(compressed);
// => Uint8Array
```
