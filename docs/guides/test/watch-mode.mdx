使用 `--watch` 标志以监听模式运行测试。

```sh terminal icon="terminal"
bun test --watch
```

---

这将在检测到文件更改时重启正在运行的 Bun 进程。速度很快。在此示例中，编辑器配置为每次按键时保存文件。
