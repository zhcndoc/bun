`ctrl+c` 快捷键会向正在运行的进程发送一个 _中断信号_。该信号可以通过监听 `SIGINT` 事件来捕获。如果你想关闭进程，必须显式调用 `process.exit()`。

```ts process.ts icon="/icons/typescript.svg"
process.on("SIGINT", () => {
  console.log("Ctrl-C was pressed");
  process.exit();
});
```
