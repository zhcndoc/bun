---
title: 将 ReadableStream 转换为 ArrayBuffer
sidebarTitle: 流到 ArrayBuffer
mode: center
---

Bun 提供了多种便捷函数，用于将 [`ReadableStream`](https://developer.mozilla.org/en-US/docs/Web/API/ReadableStream) 的内容读取为不同格式。

```ts
const stream = new ReadableStream();
const buf = await Bun.readableStreamToArrayBuffer(stream);
```

---

有关 Bun 其他 `ReadableStream` 转换函数的文档，请参见 [Docs > API > Utils](/runtime/utils#bun-readablestreamto)。
