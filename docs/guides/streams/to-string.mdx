sidebarTitle: 流转换为字符串

---

Bun 提供了许多便捷函数，用于将 [`ReadableStream`](https://developer.mozilla.org/en-US/docs/Web/API/ReadableStream) 的内容读取成不同格式。

```ts
const stream = new ReadableStream();
const str = await Bun.readableStreamToText(stream);
```

---

有关 Bun 其他 `ReadableStream` 转换函数的文档，请参见 [Docs > API > Utils](/runtime/utils#bun-readablestreamto)。
