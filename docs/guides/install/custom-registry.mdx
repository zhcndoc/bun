---
title: 覆盖 bun install 的默认 npm registry
sidebarTitle: 覆盖默认的 npm registry
mode: center
---

默认的 registry 是 `registry.npmjs.org`。可以在 `bunfig.toml` 中进行全局配置。

```toml bunfig.toml icon="settings"
[install]
# 将默认 registry 设置为字符串
registry = "https://registry.npmjs.org"

# 如有需要，设置令牌
registry = { url = "https://registry.npmjs.org", token = "123456" }

# 如有需要，设置用户名/密码
registry = "https://usertitle:password@registry.npmjs.org"
```

---

你的 `bunfig.toml` 可以引用环境变量。Bun 会自动加载 `.env.local`、`.env.[NODE_ENV]` 和 `.env` 中的环境变量。更多信息参见 [文档 > 环境变量](/runtime/environment-variables)。

```toml bunfig.toml icon="settings"
[install]
registry = { url = "https://registry.npmjs.org", token = "$npm_token" }
```
