Bun 通过 `Bun.stdout` 属性将 `stdout` 公开为一个 `BunFile`。这可以用作 [`Bun.write()`](/runtime/file-io#writing-files-bun-write) 的目标。

这段代码将文件写入 `stdout`，类似于 Unix 中的 `cat` 命令。

```ts cat.ts icon="/icons/typescript.svg"
const path = "/path/to/file.txt";
const file = Bun.file(path);
await Bun.write(Bun.stdout, file);
```

完整的 `Bun.write()` 文档请参阅 [文档 > API > 文件 I/O](/runtime/file-io#writing-files-bun-write)。
