此代码片段将字符串写入磁盘上的特定_绝对路径_。

它使用高速的 [`Bun.write()`](/runtime/file-io#writing-files-bun-write) API 来高效地将数据写入磁盘。第一个参数是_目标路径_；第二个参数是要写入的_数据_。

```ts
const path = "/path/to/file.txt";
await Bun.write(path, "Lorem ipsum");
```

---

任何相对路径都会相对于项目根目录（包含最近的 `package.json` 文件的目录）进行解析。

```ts
const path = "./file.txt";
await Bun.write(path, "Lorem ipsum");
```

---

你可以传入一个 `BunFile` 作为目标。`Bun.write()` 会将数据写入其关联的路径。

```ts
const path = Bun.file("./file.txt");
await Bun.write(path, "Lorem ipsum");
```

---

`Bun.write()` 返回写入磁盘的字节数。

```ts
const path = "./file.txt";
const bytes = await Bun.write(path, "Lorem ipsum");
// => 11
```
