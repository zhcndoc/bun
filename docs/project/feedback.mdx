使用 `bun feedback` 可直接向 Bun 团队提交反馈、错误报告和功能请求。

```bash terminal icon="terminal"
bun feedback "喜欢这个新版本！"
bun feedback report.txt details.log
echo "请添加 X 的文档" | bun feedback --email you@example.com
```

您可以通过文本参数、文件路径或管道输入的方式提供反馈。
