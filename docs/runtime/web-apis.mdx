---
title: "Web APIs"
description: "Bun 对服务器端 JavaScript 支持的 Web 标准 API"
mode: center
---

某些 Web API 在以服务器为先的运行时环境（如 Bun）中不适用，例如 [DOM API](https://developer.mozilla.org/en-US/docs/Web/API/HTML_DOM_API#html_dom_api_interfaces) 或 [History API](https://developer.mozilla.org/en-US/docs/Web/API/History_API)。但许多其他 API 在浏览器环境之外同样广泛有用；Bun 尽可能实现这些 Web 标准 API，而不是引入新的 API。

以下 Web API 得到部分或完全支持。

| URL                   | [`URL`](https://developer.mozilla.org/en-US/docs/Web/API/URL)，[`URLSearchParams`](https://developer.mozilla.org/en-US/docs/Web/API/URLSearchParams)                                                                                                                                                                                                                                                                                                                                                                                   |

| 加密                  | [`crypto`](https://developer.mozilla.org/en-US/docs/Web/API/Crypto)，[`SubtleCrypto`](https://developer.mozilla.org/en-US/docs/Web/API/SubtleCrypto)，[`CryptoKey`](https://developer.mozilla.org/en-US/docs/Web/API/CryptoKey)                                                                                                                                                                                                                                                                                                        |
