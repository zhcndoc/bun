Bun 在 `Bun` 全局对象以及多个内置模块中实现了一组原生 API。这些 API 经过高度优化，代表了实现一些常见功能的规范 “Bun 原生” 方式。

Bun 力求尽可能实现标准 Web API。Bun 主要为服务器端任务引入新的 API，这些任务在标准中尚不存在，例如文件 I/O 和启动 HTTP 服务器。在此类情况下，Bun 的方法依然构建于诸如 `Blob`、`URL` 和 `Request` 等标准 API 之上。

```ts server.ts icon="/icons/typescript.svg"
Bun.serve({
  fetch(req: Request) {
    return new Response("Success!");
  },
});
```
